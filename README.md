# monthly-budget# 💰 Personal Budget Tracker 2026
**Tarun SR — Monthly Budget Web App**

A clean, personal budget tracker built as a static web app. No backend needed. Runs entirely in the browser and stores data locally. Deploy for free via GitHub Pages.

---

## ✨ Features

- 📅 **June → December 2026** month-by-month tracking
- 💵 Income tracking (Salary + Additional Income)
- 🔒 Fixed expenses pre-filled (Home, PG, Gold, MF/ETF, PPF, Wifi, Phone)
- 📝 Free-form variable expenses — add anything, any month
- 📈 Savings tracker with **auto-carry** — enter balance once in June, rest auto-calculates
- ✅ Bank balance verification — checks if your app balance matches the calculated closing balance
- 📊 Full year overview table
- ⬇ Export to CSV anytime
- ⬆ Import from CSV to restore data
- 💾 Auto-saves to browser localStorage

---

## 🚀 Deploy on GitHub Pages (3 steps)

1. **Create a new GitHub repository** (e.g. `budget-2026`)
2. **Upload** `index.html` and `README.md` to the repo
3. Go to **Settings → Pages → Source → Deploy from branch → main → / (root)**

Your app will be live at:
```
https://YOUR-USERNAME.github.io/budget-2026/
```

---

## 📱 How to Use

### First Time (June)
1. Open the app
2. A banner will ask for your **current bank balance** — enter it once
3. This becomes your June opening balance

### Every Month
1. Click the month tab (June, July, etc.)
2. Update **salary** and **additional income** if changed
3. Add your **variable expenses** (groceries, EMI, dining, whatever you spent)
4. At month end, enter your **actual bank balance** to verify it matches
5. Click **💾 Save**

### Export / Backup
- Click **⬇ Export CSV** to download your full data as a CSV
- Upload that CSV to your GitHub repo for backup
- Click **⬆ Import CSV** to restore data on any device
